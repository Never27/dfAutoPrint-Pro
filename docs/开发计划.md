## 风险评估

| 风险 | 概率 | 影响 | 缓解措施 |

| .NET 8 EOL | 低 | 中 | 计划 2026-11 前迁移到 .NET 10 |

| Windows 打印机 API 变化 | 低 | 中 | 使用 WMI 稳定接口 |
| 用户环境差异 | 中 | 中 | 完善错误处理和日志 |

## 待解决问题

1. **安装包格式**：当前仅提供 PS 脚本，需制作 MSI/EXE 安装包
2. **自包含发布**：.NET 10 SDK NuGet 问题导致无法 self-contained publish，需迁移到正式环境
3. **截图和演示**：需要制作软件运行截图和演示视频
4. **CI/CD**：需要配置 GitHub Actions 自动构建和发布
