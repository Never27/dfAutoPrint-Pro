## 什么是 PdfAutoPrint Pro？

PdfAutoPrint Pro 是一款专为 Windows 平台打造的**虚拟 PDF 打印机管理器**。它在您的系统中注册一个虚拟打印机，当您从任何应用程序（Word、Excel、浏览器、CAD、图像软件等）执行"打印"操作时，PdfAutoPrint Pro 会自动拦截打印任务，将其转换为高质量 PDF 文件并保存到您指定的位置。

与普通 PDF 打印机不同，PdfAutoPrint Pro 提供了一整套**智能化的文件管理、命名规范和输出配置**能力，让 PDF 输出流程真正实现自动化和标准化。

---

## 它解决了什么问题？

在日常工作中，将文档打印为 PDF 是一个高频需求，但现有的方案存在诸多痛点：

| 痛点 | 现有方案的问题 | PdfAutoPrint Pro 的解决方式 |
|------|-------------|---------------------------|
| **命名混乱** | 每次都要手动输入文件名，容易重复 | 智能文件名模板，自动填充日期、来源等信息 |
| **重复保存** | 文件名冲突时无提示直接覆盖 | 三种冲突策略，确保文件安全 |
| **批量中断** | 打印大量文件时一个失败全部中止 | 失败自动跳过，不中断后续任务 |
| **状态未知** | 打印是否成功无从得知 | 系统托盘通知 + 详细日志 |
| **质量不可控** | 输出 PDF 无法调节色彩和分辨率 | 彩色/灰度、矢量/图片、DPI 可配置 |
| **缺少水印** | 需要额外工具为 PDF 添加水印 | 内置水印引擎，文字/图片水印随选 |
| **安装繁琐** | 需要分别下载安装各种依赖 | 一键安装脚本，全自动配置环境 |

## 核心优势

### 零外部依赖

整个应用基于 .NET 8 WPF 原生构建，**不依赖任何第三方 NuGet 包**。除了运行时必需的 .NET 8 Desktop Runtime 和底层使用的 GhostScript（由安装脚本自动处理），项目没有引入任何额外的外部库。这意味着：

- 安装包体积极小
- 无供应链安全风险
- 不依赖不可控的第三方更新
- 长期维护无忧

### 轻量级

- 内存占用极低，常驻系统托盘不打扰
- 启动速度快，开机自启无明显感知
- 不影响系统打印性能

12 项核心功能覆盖从打印到输出的每一个环节，详见[功能介绍](./功能介绍.md)。
