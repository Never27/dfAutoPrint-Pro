## 1. 自动/手动保存切换

### 功能说明

控制 PDF 文件生成时的保存行为，支持两种模式：

- **自动保存模式**：PDF 生成后直接存入预设目录，无需任何弹窗确认
- **手动模式**：每次打印弹出文件保存对话框，由用户选择保存位置和文件名

### 如何配置

在系统配置面板的"保存设置"中切换，该设置可**按打印机独立配置**。

```
自动保存模式：开启后，文件自动存入默认目录
手动模式：关闭后，每次弹出保存对话框
```

### 适用场景

| 场景 | 推荐模式 |
|------|---------|
| 批量打印大量文档 | 自动保存 |
| 需要精确命名和选择路径 | 手动模式 |
| 自动化工作流集成 | 自动保存 |

## 2. 智能文件名模板

### 功能说明

支持使用占位符动态生成文件名，无需每次手动输入。系统在生成 PDF 时自动解析占位符，替换为实际信息。

### 支持的占位符

| 占位符 | 说明 | 示例输出 |
|--------|------|---------|
| `<PrintJobName>` | 打印任务名称（即文档标题） | `季度报告` |
| `<PrintJobAuthor>` | 打印文档作者 | `张三` |
| `<DateTime:format>` | 当前日期时间（自定义格式） | `2026-06-05` |
| `<Counter>` | 自增计数器（全局） | `001`、`002` |
| `<PrinterName>` | 当前打印机名称 | `PdfAutoPrint_办公` |

### 配置示例

```
模板：<DateTime:yyyyMMdd>_<PrintJobName>_<Counter:D3>
结果：20260605_季度报告_001.pdf

模板：<PrintJobAuthor>/<DateTime:yyyy-MM>/<PrintJobName>
结果：张三/2026-06/季度报告.pdf
```

### 占位符嵌套规则

- 可使用 `/` 在模板中创建子目录
- `<DateTime:format>` 中的 `format` 遵循 .NET 日期格式化规则
- `<Counter:D3>` 中的 `D3` 表示 3 位数零填充

### 小贴士

- 如果文档标题包含 `*原始标题*`（如 Word 临时文件名），系统会自动过滤
- 建议将日期放在文件名最前面，便于按时间排序

## 3. 命名冲突处理

### 功能说明

当生成的文件名与已有文件重名时，提供三种处理策略，安全可控。

### 三种策略

| 策略 | 行为 |
|------|------|
| **覆盖（Overwrite）** | 直接替换已有文件，不保留原文件 |
| **递增编号（Increment）** | 在文件名后添加序号，如 `报告(1).pdf`、`报告(2).pdf` |
| **时间戳备份（Backup）** | 将已有文件重命名为 `报告_20260605_143052.bak`，再保存新文件 |

### 如何配置

在配置面板的"冲突处理"区域，每个打印机可单独选择策略。

### 适用场景

| 策略 | 适用场景 |
|------|---------|
| 覆盖 | 临时文件、每次需要最新版本的场景 |
| 递增编号 | 需要保留所有历史版本的场景 |
| 时间戳备份 | 需要追溯任意时间点版本的合规场景 |

## 4. 系统托盘通知

### 功能说明

每次打印任务完成后，通过 Windows 系统托盘弹出气泡通知，告知用户任务结果。

### 通知类型

| 类型 | 图标 | 场景 |
|------|------|------|
| 成功通知 | 信息图标 | PDF 生成成功，显示文件名和保存路径 |
| 失败通知 | 警告图标 | 打印失败，显示失败原因摘要 |

### 如何配置

- 在设置中可开启/关闭通知
- 可配置成功通知是否显示（失败通知默认始终显示）
- 点击通知可快速打开所在文件夹

### 小贴士

- 批量打印时，通知会合并显示摘要，避免消息轰炸
- 可在 Windows 通知设置中进一步控制通知优先级

## 5. 日志保留策略

### 功能说明

详细记录每次打印任务的信息，包括时间、来源应用、文件名、结果等，支持灵活保留策略。

### 日志内容

- 任务创建时间
- 源应用名称（如 Word、Chrome）
- 输出文件路径
- 处理耗时
- 成功/失败状态及错误详情

### 保留策略

| 策略 | 说明 |
|------|------|
| **保留 N 天** | 超过指定天数的日志自动清理，节省磁盘 |
| **永久保留** | 日志永不自动删除，适合合规审计需求 |

### 如何配置

在系统配置的"日志设置"中，设置保留天数（0 表示永久保留）和最大日志文件大小。

### 适用场景

- 企业内部审计，需要追溯每份 PDF 的生成历史
- 排查打印失败原因
- 统计打印频率和使用习惯

## 6. 失败自动跳过

### 功能说明

在批量打印场景下，如果某一项任务失败（如文件损坏、权限不足），系统自动跳过该任务，继续处理后续任务，不会中断整个批量流程。

### 工作原理

1. 打印队列中进入新任务
2. 系统尝试处理该任务
3. 如果失败，记录错误日志 + 发送失败通知
4. 自动清除该任务，继续处理队列中的下一项

### 如何配置

默认开启，可在设置中关闭（关闭后，失败会暂停队列等待用户干预）。

### 小贴士

- 失败的任务会记录在日志查看器中，可集中查看和排查
- 批量打印前建议先测试 1-2 个文件确保配置正确

## 7. WPF 桌面界面

### 功能说明

提供友好的 Windows 原生桌面界面，包含四个核心模块。

### 四大模块

#### 7.1 仪表盘（Dashboard）

- 实时显示今日打印统计
- 系统打印机状态一览
- 快捷操作入口
- 最近打印任务列表

#### 7.2 打印机管理器（Printer Manager）

- 查看所有系统打印机（含实体打印机）
- 虚拟打印机的添加、删除、重命名
- 打印机属性配置入口
- 打印队列监控
- Spooler 服务状态和控制

#### 7.3 日志查看器（Log Viewer）

- 按时间、状态、打印机筛选日志
- 日志详情查看
- 导出日志为 CSV
- 批量清除旧日志
- 搜索和关键字高亮

#### 7.4 任务历史（Job History）

- 历史任务列表和状态
- 按日期范围筛选
- 快速打开输出文件位置
- 重新打印失败任务

## 8. 输出质量控制

### 功能说明

精细控制 PDF 输出质量，满足不同场景的清晰度和文件大小需求。

### 配置项

| 配置项 | 选项 | 说明 |
|--------|------|------|
| **色彩模式** | 彩色 / 灰度 | 灰度模式可显著减小文件体积 |
| **输出模式** | 矢量 / 图片 | 矢量保持文字可选中，图片模式适合排版固定场景 |
| **DPI** | 72 / 150 / 300 / 600 / 1200 | 控制图片模式下渲染分辨率 |

### 推荐配置

| 场景 | 色彩 | 模式 | DPI |
|------|------|------|-----|
| 日常文档存档 | 彩色 | 矢量 | — |
| 网页保存 | 彩色 | 矢量 | — |
| CAD 图纸输出 | 灰度 | 图片 | 600 |
| 设计稿交付 | 彩色 | 图片 | 300 |
| 档案归档（最小体积） | 灰度 | 矢量 | — |

### 小贴士

- 矢量模式生成的 PDF 中文字可被选中和搜索
- 高 DPI + 图片模式会产生较大文件，仅在需要时使用
- 彩色模式和灰度模式的切换通过 GhostScript 参数控制

## 9. 一键安装工具

### 功能说明

通过单个 PowerShell 脚本完成全套环境搭建，无需手动下载和配置各组件。

### 自动完成的任务

```
1. 检测系统环境（Windows 版本、架构）
2. 检查 .NET 8 Desktop Runtime（缺失则下载安装）
3. 下载并配置 GhostScript
4. 安装虚拟 PDF 打印机驱动程序
5. 创建虚拟打印机并注册
6. 注册 PdfAutoPrint Pro 应用
7. 创建开始菜单快捷方式
8. 配置开机自启（可选）
```

### 使用方式

```powershell
# 右键"以管理员身份运行 PowerShell"
cd C:\path\to\PdfAutoPrint.Pro\scripts
.\install.ps1
```

### 小贴士

- 需要管理员权限
- 安装过程中自动跳过已存在的组件
- 支持静默安装模式（`.\install.ps1 -Silent`）
- 安装日志保存在 `%TEMP%\PdfAutoPrint_Install.log`

## 10. 打印机管理

### 功能说明

集中管理 Windows 系统中的所有打印机，特别是虚拟 PDF 打印机。

### 管理功能

| 功能 | 说明 |
|------|------|
| **重命名** | 修改虚拟打印机显示名称 |
| **查看系统打印机** | 列出系统中所有打印机及其状态 |
| **重启 Spooler** | 一键重启 Print Spooler 服务，解决常见打印故障 |
| **设置默认打印机** | 将虚拟打印机设为系统默认打印机 |
| **查看打印队列** | 查看当前打印机的待处理队列和任务状态 |
| **删除打印机** | 移除不再需要的虚拟打印机 |
| **添加新打印机** | 创建额外的虚拟打印机（见功能 12） |

### 常见操作

```
打印机卡住 → 点击"重启 Spooler"
需要临时切换默认 → 右键"设为默认打印机"
排查问题 → 点击"查看打印队列"
```

## 11. 水印支持

### 功能说明

在生成的 PDF 文件上自动添加水印，支持文字水印和图片水印两种类型。

### 水印类型

#### 文字水印

| 属性 | 说明 |
|------|------|
| 文字内容 | 支持多行文字，可使用文件名等变量 |
| 字体 | 系统字体选择，支持大小、颜色配置 |
| 透明度 | 0-100% 可调 |
| 旋转角度 | 0-360 度，常用对角线 45 度 |
| 位置 | 九宫格位置 + 像素微调（绝对定位） |
| 间距 | 平铺模式下的行列间距 |

#### 图片水印（图片盖章）

| 属性 | 说明 |
|------|------|
| 图片来源 | 支持 PNG、JPG、BMP 格式 |
| 缩放 | 按比例或按像素缩放 |
| 透明度 | 0-100% |
| 位置 | 九宫格 + 偏移，支持自动居中 |

### 页面跳过

可配置跳过的页面范围：

```
跳过第一页：封面不加水印
仅奇数/偶数页：双面打印对应页面
指定页码范围：如 2-5 页
```

### 配置示例

```
场景：公司内部文件归档
水印类型：文字
内容："内部资料 仅供 XX 公司使用"
透明度：15%
角度：45°
位置：居中
跳过：第 1 页（封面）
```

### 小贴士

- 建议透明度控制在 10%-30%，既可见又不影响阅读
- 图片水印建议使用 PNG 格式以支持透明通道
- 文字水印支持中文字体，建议使用系统自带宋体/微软雅黑

## 12. 多打印机配置

### 功能说明

支持创建多个虚拟 PDF 打印机，每台打印机拥有独立的完整配置。不同打印机可用于不同工作场景，互不干扰。

### 典型用法

| 打印机名称 | 配置 | 用途 |
|-----------|------|------|
| `PdfAutoPrint_办公` | 自动保存、矢量、无印章 | 日常文档转换 |
| `PdfAutoPrint_归档` | 递增编号、灰度、时间戳备份 | 历史文档归档 |
| `PdfAutoPrint_印章` | 公司章水印、彩色 | 正式合同生成 |
| `PdfAutoPrint_设计` | 高DPI、图片模式 | 设计稿输出 |

### 独立配置项

每台打印机可独立配置：

- 输出目录
- 文件命名模板
- 冲突处理策略
- 水印设置
- 输出质量（色彩/DPI/模式）
- 通知行为

### 如何管理

在打印机管理器界面：
- 点击"添加新打印机"创建
- 右键打印机名称进行配置
- 拖拽排序打印机列表

### 小贴士

- 建议按使用场景命名打印机，方便识别
- 打印机配置可导出/导入，便于团队统一配置
- 最多支持创建 10 台虚拟打印机
