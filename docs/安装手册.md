## 安装前检查清单

在安装之前，请逐项确认以下内容：

- [ ] 操作系统为 Windows 7/10/11 64 位版本
- [ ] 当前用户具有管理员权限
- [ ] 系统未安装过旧版 PdfAutoPrint（如有请先卸载）
- [ ] 关闭杀毒软件的实时防护（安装完成后可重新开启）
- [ ] 系统 Print Spooler 服务处于运行状态

### 如何检查 Print Spooler 状态

1. 按 `Win + R`，输入 `services.msc`，回车
2. 在服务列表中找到 **Print Spooler**
3. 确认状态为"正在运行"，启动类型为"自动"

如果未运行：
```powershell
# 以管理员身份运行 PowerShell
Start-Service Spooler
Set-Service Spooler -StartupType Automatic
```

## 一键安装指南

### 步骤 1：获取安装包

解压 PdfAutoPrint Pro 安装包到任意目录，例如：

```
C:\Program Files\PdfAutoPrint.Pro\
```

### 步骤 2：运行安装脚本

**以管理员身份运行 PowerShell**：

1. 按 `Win + X`，选择 **Windows PowerShell（管理员）** 或 **终端（管理员）**
2. 切换到脚本目录：

```powershell
cd "C:\Program Files\PdfAutoPrint.Pro\scripts"
```

3. 执行安装：

```powershell
.\install.ps1
```

### 步骤 3：安装过程

脚本将依次执行以下操作，请耐心等待：

```
[1/7] 检测系统环境...                     ✓ 完成
[2/7] 检查 .NET 8 Desktop Runtime...      ✓ 已安装 / → 正在安装...
[3/7] 下载 GhostScript...                  → 正在下载...
[4/7] 安装 GhostScript...                  ✓ 完成
[5/7] 安装虚拟打印机驱动...                ✓ 完成
[6/7] 创建虚拟打印机...                    ✓ 完成
[7/7] 注册应用并创建快捷方式...            ✓ 完成

安装完成！PdfAutoPrint Pro 已就绪。
```

### 安装选项

```powershell
# 静默安装（不弹出交互提示）
.\install.ps1 -Silent

# 指定安装目录
.\install.ps1 -InstallPath "D:\Tools\PdfAutoPrint"

# 安装额外虚拟打印机
.\install.ps1 -Printers @("办公", "归档", "合同")

# 跳过 GhostScript（如已安装）
.\install.ps1 -SkipGhostScript

# 查看帮助
.\install.ps1 -Help
```

## 手动安装（高级）

如果您需要手动安装各个组件，请按以下步骤操作：

### 1. 安装 .NET 8 Desktop Runtime

1. 访问 [.NET 下载页面](https://dotnet.microsoft.com/download/dotnet/8.0)
2. 下载 **.NET Desktop Runtime 8.0.x (Windows x64)**
3. 运行安装程序，全部默认选项完成安装
4. 验证安装：

```powershell
dotnet --list-runtimes
```

### 2. 安装 GhostScript

1. 访问 [GhostScript 下载页面](https://ghostscript.com/releases/gsdnld.html)
2. 下载 **AGPL Release** 64 位版本
3. 安装到默认路径或自定义路径
4. 将 GhostScript 的 `bin` 目录添加到系统 PATH 环境变量
5. 验证安装：

```powershell
gswin64c --version
```

### 3. 安装虚拟打印机驱动

1. 打开"控制面板" → "设备和打印机"
2. 点击"添加打印机"
3. 选择"我所需的打印机未列出"
4. 选择"添加本地打印机或网络打印机"
5. 使用现有端口：`FILE: (打印到文件)`
6. 厂商选择"Generic"，打印机选择"Generic / Text Only"
7. 命名为"PdfAutoPrint Pro"
8. 不共享打印机，完成添加

### 4. 配置 PdfAutoPrint Pro

```powershell
# 注册虚拟打印机关联
.\PdfAutoPrint.Pro.exe --register

# 启动应用
.\PdfAutoPrint.Pro.exe
```

## 首次启动

### 启动应用

- 双击桌面 **PdfAutoPrint Pro** 图标
- 或从开始菜单启动

### 初始配置向导

首次启动时将引导您完成基础配置：

1. **选择语言** — 默认简体中文
2. **输出目录** — 设置 PDF 默认保存位置
3. **保存模式** — 自动保存 / 手动选择
4. **文件命名模板** — 默认模板或自定义
5. **通知偏好** — 是否开启托盘通知
6. **开机启动** — 是否随 Windows 自启

### 快速验证

完成向导后，进行快速验证：

1. 打开任意文档（如记事本）
2. 按下 `Ctrl + P` 打印
3. 选择打印机 **PdfAutoPrint Pro**
4. 点击"打印"
5. 查看输出目录是否生成了 PDF 文件
6. 查看系统托盘是否有通知弹出

### 预期结果

- 输出目录中出现 PDF 文件
- 系统托盘显示 PdfAutoPrint Pro 图标
- 托盘弹出"打印成功"通知
- 主界面仪表盘计数 +1

## 常见问题排查

### Q1：安装脚本提示"无法获取管理员权限"

**原因**：未以管理员身份运行 PowerShell

**解决**：
1. 右键 PowerShell 图标
2. 选择"以管理员身份运行"
3. 重新执行安装脚本

### Q2：Print Spooler 服务无法启动

**原因**：打印服务可能被禁用或文件损坏

**解决**：
```powershell
# 清除打印队列
net stop spooler
del /Q /F "%systemroot%\System32\spool\PRINTERS\*"
net start spooler
```

如果仍失败：
```powershell
# 重置打印系统
sc config spooler depend= RPCSS
net start spooler
```

### Q3：GhostScript 下载失败

**原因**：网络问题或防火墙阻止

**解决**：
- 手动下载 GhostScript 安装包
- 使用 `-SkipGhostScript` 参数跳过自动下载
- 检查代理设置或临时关闭防火墙

### Q4：虚拟打印机无法创建

**原因**：驱动文件缺失或权限不足

**解决**：
1. 确认以管理员身份运行
2. 关闭杀毒软件的"驱动保护"功能
3. 手动检查驱动文件是否存在于安装目录

### Q5：PDF 生成后是空白文件

**原因**：GhostScript 路径未正确配置

**解决**：
1. 打开 PdfAutoPrint Pro
2. 进入"设置" → "高级"
3. 手动指定 GhostScript 可执行文件路径
4. 通常位于：`C:\Program Files\gs\gs10.x.x\bin\gswin64c.exe`

### Q6：打印后无任何反应

**原因**：应用未启动或服务异常

**解决**：
1. 确认 PdfAutoPrint Pro 在运行（检查系统托盘图标）
2. 查看 Windows 打印队列是否卡住
3. 在应用中点击"重启 Spooler"
4. 重启 PdfAutoPrint Pro

### Q7：文件名中的中文显示为乱码

**原因**：系统区域设置或编码问题

**解决**：
1. 确认 Windows 系统区域设置为"中文（简体，中国）"
2. 控制面板 → 区域 → 管理 → 更改系统区域设置
3. 勾选"Beta 版：使用 Unicode UTF-8"

### 日志位置

如需深入排查，请查看日志文件：

```
%APPDATA%\PdfAutoPrint.Pro\logs\
安装日志：%TEMP%\PdfAutoPrint_Install.log
```

## 卸载指南

### 自动卸载

运行卸载脚本：

```powershell
# 以管理员身份运行
.\scripts\uninstall.ps1
```

脚本将自动：
1. 停止 PdfAutoPrint Pro 运行
2. 删除虚拟打印机
3. 移除打印机驱动
4. 删除应用文件（可选保留配置）
5. 清除注册表项
6. 删除开始菜单快捷方式

### 手动卸载步骤

如需手动卸载：

#### 1. 删除虚拟打印机

控制面板 → 设备和打印机 → 右键 PdfAutoPrint Pro → 删除设备

#### 2. 删除打印机驱动

```powershell
# 以管理员运行
printui.exe /s /t2
# 在"驱动程序"选项卡中找到相关驱动 → 删除
```

#### 3. 卸载应用

控制面板 → 程序和功能 → 卸载 PdfAutoPrint Pro

或直接删除安装目录。

#### 4. 清理残留

删除以下目录（如果存在）：

```
%APPDATA%\PdfAutoPrint.Pro\
%LOCALAPPDATA%\PdfAutoPrint.Pro\
```

### GhostScript 卸载

如不再需要 GhostScript，通过控制面板 → 程序和功能 → 卸载 GhostScript。
